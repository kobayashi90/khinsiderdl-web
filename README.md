A Video Game Music Streaming Web App. 

## Install

- In the project folder:

```bash
npm install
```

## Run locally

```bash
npm run dev
```

Open `http://localhost:3000`.

## Build + run (production)

```bash
npm run build
npm run start
```

## Optional env (playlist sharing)

- `PLAYLIST_SHARE_DATA_DIR`: absolute or relative directory for shared-playlist JSON files.
  - default: `./data/playlist-shares`
