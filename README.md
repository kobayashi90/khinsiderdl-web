## Install

- Install **Node.js 18+** (Node 20+ recommended)
- In the project folder:

```bash
npm install
```

## Run locally

```bash
npm run dev
```

Open `http://localhost:3000`.

## Build + run (production)

```bash
npm run build
npm run start
```

## API endpoints

- `/api/search?q=...`
- `/api/album?url=...`
- `/api/resolve?url=...`
- `/api/image?url=...`
- `/api/download?url=...`
